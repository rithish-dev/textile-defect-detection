# Textile Defect Detection 🧵

Real-time fabric defect detection system for garment factories using YOLOv8 and computer vision.

Built specifically for the Tirupur textile industry — where defect checking is still done manually.

## Problem
Indian garment factories lose 3–8% of revenue to defective fabric shipments.
Manual inspection is slow, inconsistent, and expensive at scale.

## Solution
A camera-based AI system that detects fabric defects in real time on the production line.

## Tech Stack
- YOLOv8 (Ultralytics)
- Python, OpenCV
- Dataset: Fabric Defect Detection (Kaggle)
